# Projeto #4- Efeitos Sonoros
Toca um efeito sonoro conforme a tecla pressionada.

Os sons são gerados usando um módulo MP3 DFPlayer Mini. O código
não trata as respostas do módulo. Os arquivos com os sons devem
estar no diretório mp3 do micro SD e terem nomes 0001.mp3 a 0006.mp3.

As seis teclas são ligadas a uma única entrada analógica.

Os sons usados para teste estão em Sons:

* 0001.mp3 https://www.freesoundeffects.com/free-track/bark-89451/
* 0002.mp3 https://www.freesoundeffects.com/free-track/rain-thunder-468881/
* 0003.mp3 https://www.freesoundeffects.com/free-track/elephant--466267/
* 0004.mp3 https://www.freesoundeffects.com/free-track/airplanes-89251/
* 0005.mp3 https://www.freesoundeffects.com/free-track/car-horn-x-466318/
* 0006.mp3 https://www.freesoundeffects.com/free-track/applause2-426807/
