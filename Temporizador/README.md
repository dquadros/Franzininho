# Projeto #1 - Temporizador
Aciona um relê pelo tempo definido por um potenciômetro.

Após selecionar o tempo no potenciômetro, apertar e soltar o botão para o
relê ser acionado. O tempo máximo usado é de 10 minutos.

É utilizado um "2 relay module"

O potenciômetro deve ser linear, com valor entre 1K e 100K.
